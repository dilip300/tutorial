Web Development With Syro

Syro is a router for web development with Ruby. It offers a very simple and scalable API that encourages modularity and makes it easy to create applications big and small without sacrificing maintainability.

Getting started

Syro is a gem, so you can install it with the following command: 

  gem install syro

Now you can create your first Syro app! Remember that you can always go back to this intro by clicking on the Syro logo.
